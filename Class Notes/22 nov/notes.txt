CSS: Cascading Style Sheet

Syntax:

Always write css code in the head 
<style>
	selector{
	   property-name:value;
		
	}
</style>

selector will select an element on which style has to be applied
font-size
background-color


Types of selectors in css:
1. Tag selector: name of the tag, it will select all the matching tag by name
		 tag-name{
		}
2. Id selector: id of the element will be the selector
		#id-value{}

3. Class selector:  identify multiple tags commonly 

4. Universal selector: *{}


id>class>tag-name>*
